## Prerequisites

In order to abode compatibility issues please try to use this environment setup

- NodeJS Version 16.15.0
- NPM Version 8.5.5

## Instalation

To install the dependencies run

```bash
  npm install
```

## Execution

To execute the solution, first **is mandatory to create a .env file with the EMAIL key inside it**. Follow the **.env.example** example file and add an email value.
Then run

```bash
  npm start
```

## Test

To execute the tests run

```bash
  npm test
```
